# Support Plan – Tutorio

## Issue Reporting Process
Users can report issues through:
- Email: support@tutorio.com
- GitHub Issues page

## Response Time
- Critical issues: within 24 hours
- Minor issues: within 48–72 hours

## Support Workflow
1. User reports issue
2. Team reviews and assigns priority
3. Developer fixes issue
4. QA tests fix
5. Update deployed system

## Maintenance
- Weekly system checks
- Regular updates and bug fixes
